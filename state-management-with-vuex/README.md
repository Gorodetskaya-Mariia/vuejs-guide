# state-management-with-vuex

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
npm install --save vuex,

Vuex Documenation:https://vuex.vuejs.org/en/
