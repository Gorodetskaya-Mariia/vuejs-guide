# validation

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# install vuex
npm install --save vuex

# install vue-router
npm install --save vue-router

# install axios
npm install --save axios

# install vuelidate
npm install --save vuelidate
```
